* 프로그램 실행시 필요한 프로그램
1. MariaDB 10.3.16
2. Anaconda3-2019.03-Windows-x86_64
3. 크롬 드라이버(크롬에 맞는 버전으로 설치할 것!)

* 프로그램 실행시 설치 해야하는 목록
1. pip install flask
2. pip install pymysql
3. pip install BeautifulSoup
4. pip install urllib
5. pip install selenium
8. pip install soynlp
9. pip install wordcloud
10. pip install matplotlib
11. pip install PIL

* MariaDB의 데이터는 데이터 내보내기 함
경로는 바탕화면으로 세팅

* MariaDB의 인코딩 상태
utf8mb4
